Upload a use-case bundle
A bundle is a slug + manifest.yaml + ontology.ttl + data.ttl. The slug becomes the directory name under use_cases/ and must be lower-case alphanumeric (with - or _). To include LLM agents, declare them in the manifest's agents: block — see the agent template.
